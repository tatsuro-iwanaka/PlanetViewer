#PlanetViewer: Planetary Observation Planner.

Simulate the disk views of Mercury, Venus, Mars and Jupiter from the Earth.

Range of calculation date: Jan 1, 1970 to Dec 31, 2120

Observation sites:
  - IRTF, Mauna Kea
  - T60, Haleakala
  - NAOJ, Miatka
  - Tohoku U., Sendai
  - R-CCS, Kobe
  - ALMA, Chile
